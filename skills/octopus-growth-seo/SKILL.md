# Octopus Growth SEO

Improve qualified industrial demand, not vanity traffic.

## Strategy

1. Confirm the active domain, brand name, audience, service scope, and primary conversion from current files or live sources.
2. Map search intent to real V1 capabilities: industrial sourcing, supplier qualification, RFQs, local capacity, corridor coverage, verification, and operational coordination.
3. Prefer one strong public page per distinct user intent. Do not manufacture thin location pages or unsupported claims.
4. Keep the primary CTA aligned with a controlled RFQ or private-console workflow.

## Implementation

- Provide accurate title, description, canonical URL, robots policy, sitemap, Open Graph, Twitter card, and favicon assets.
- Add JSON-LD only when every field is supported by public evidence. Prefer `Organization`, `WebSite`, and applicable `Service`; never invent ratings, reviews, prices, addresses, or clients.
- Use French as the primary public language unless the current product explicitly adds another locale.
- Keep headings semantic, internal anchors stable, images descriptive, and performance budgets compatible with mobile networks.
- Never expose private supplier records, internal scores, credentials, or non-public operational evidence for SEO.

## Validation

Inspect rendered metadata and structured data, canonical consistency, indexability, public copy restrictions, link targets, image dimensions, Core Web Vitals signals, and mobile layout. Distinguish verified production facts from local-only evidence. Define conversion events before recommending experiments.
